# [Kenzie Hub](https://kenzie-hub.raphaelgs96.vercel.app/)

<p align="center"> <img src="https://i.imgur.com/UnrDXRv.png" /> </p>

## Resumo
Kenzie Hub é um acervo de programadores que cursaram o [Kenzie Academy Brasil](https://kenzie.com.br/), onde eles podem botar as suas informações profissionais e seus projetos.

## Tecnologias
- React & Redux
- Styled Components
- Material UI
- React Swipable Views
